# Ohi ボイスプレイヤー

おひつじめぇの音声クリップを再生するウェブベースのプレイヤーです。

## 機能

- クリック一つで音声クリップを再生
- 音声クリップを検索する機能
- デスクトップとモバイル両方に対応したレスポンシブデザイン
- BootstrapベースのUIとFontAwesomeアイコン
- 音声再生時のエラーハンドリング

## 使い方

[index.html](index.html)をウェブブラウザで開くだけで使用できます：

- ボタンをクリックして音声クリップを再生
- 検索ボックスを使って利用可能な音声クリップをフィルタリング
- 提出フォームから新しい音声クリップを提案
